# Desarrollo Web - Proyecto 1
**Para BackEnd** 
1. Instalar Dependencias:
  1. composer install
  2. npm install
  3. crear base de datos laravel: ver archivo con SQl
  4. crear archivo .env
  5. generar llave : php artisan key:generate
  6. Ejecutar Migraciones: php artisan migrate
 
**Front end**
1. npm install 
2. ng serve -o
